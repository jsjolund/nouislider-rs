# nouislider-rs

A [wasm-bindgen](https://github.com/rustwasm/wasm-bindgen)
wrapper for
[noUiSlider](https://refreshless.com/nouislider/)

## Usage

```toml
[dependencies]
nouislider = "0.1"
```

## Example

See:

- Usage in yew component: <https://github.com/jsjolund/nouislider-rs/tree/main/examples/yew-component>
